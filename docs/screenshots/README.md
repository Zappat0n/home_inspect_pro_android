# Play Store Screenshots

## Requirements

- **Phone**: 8 screenshots minimum per language
- **Tablet**: 7 screenshots optional per language
- Recommended phone resolution: **1080 x 2400**
- Recommended tablet resolution: **1600 x 2560**
- Format: PNG or JPEG
- Max file size: 8 MB each

## Required Screenshots (Phone)

Capture these 8 screens for each language:

1. **Inspection list (dashboard)** — Show the main dashboard with a list of inspections
2. **New inspection form** — Show the form for starting a new inspection
3. **Checklist items by category** — Show categorized checklist items (roof, plumbing, etc.)
4. **Photo capture** — Show a checklist item with a photo attached
5. **Defect with comment** — Show a marked defect with an inspector comment
6. **Signature capture** — Show the digital signature screen
7. **PDF report preview** — Show the generated PDF report
8. **Subscription page** — Show the subscription/billing page with pricing

## Directory Structure

```
docs/screenshots/
  en/
    phone/       # 8 English phone screenshots
    tablet/      # 7 English tablet screenshots (optional)
  es/
    phone/       # 8 Spanish phone screenshots
    tablet/      # 7 Spanish tablet screenshots (optional)
```

## Naming Convention

Use zero-padded numbers for ordering:

- `01_dashboard.png`
- `02_new_inspection.png`
- `03_checklist.png`
- `04_photo_capture.png`
- `05_defect_comment.png`
- `06_signature.png`
- `07_pdf_report.png`
- `08_subscription.png`

## Tips

- Use realistic data (not "Lorem ipsum")
- Highlight key features with callout text or annotations
- Ensure all text matches the language of the screenshot
- Use a consistent device frame across all screenshots
- Test on a real device or emulator at the recommended resolution
